Communityvideo

ComunityVideo Es un plugin jQuery dedicado a la personalizacion de videos html5 basado en estilos css y css3, el plugin es actualizado por la comunidad, siendo Cristian Mari quien aporta la primera version de este plugin. Es openSoucer por lo que los usuarios podran hacer uso libre de él.

View project onGitHub

Bienvenido a la documentacion

Instalacion en 3 simples pasos

  1. Primero descarga y descomprime el plugin.
  2. Coloca este codigo dentro de tu documento
<link type="text/css" rel="stylesheet" href="css/CommunityVideo.css">
<script type="text/javascript" src="js/CommunityVideo.js">
  1. Para inicializar de forma global el plugin pega este codigo al final de tu documento
<script type="text/javascript">
    $('video').CommunityVideo();
</script>

Opciones del plugin 

<script type="text/javascript">
$('video').CommunityVideo({

    // opciones por default
    theme       : 'youtube',
    controllers : ['progress','playToggle','volume','time','fullscreen'],
    width       : '480px',
    height      : '320px',

});
</script>
  • La opcion theme corresponde al tipo de estilo que se procedera a usar para los videos.
  • La opcion controllers define los tipos de controles a mostrar y su posicion.
  • Las opciones width y height corresponde al tamaño que se usaran para los videos.

Preguntas Frecuentes

¿ En caso de querer distintos tamaños ?

Para lograr esto usted debera especificar en los videos con los atributos data-video-width y data-video-height los tamaños deseados para cada video, de lo contrario se usaran los obtenidos por default.
